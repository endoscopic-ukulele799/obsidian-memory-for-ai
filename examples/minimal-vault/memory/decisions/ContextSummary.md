# ContextSummary — Decisions

This folder records durable decisions whose rationale should survive the session where they were made. Not every choice — only the ones where future-me or the AI would ask "why did we do it this way?"

## When to create a decision
- Choosing a data format or tool for a long-term project
- Changing the structure of the memory system itself
- Deciding on a research methodology that will affect months of work
- Rejecting a reasonable alternative that someone might suggest again

## When NOT to create a decision
- Routine choices (which file to edit, minor formatting)
- Temporary workarounds
- Anything the code or data itself makes obvious
