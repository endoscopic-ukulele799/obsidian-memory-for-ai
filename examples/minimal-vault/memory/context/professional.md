# Professional context

## Role
- Senior conservator, Gemäldegalerie, Berlin
- Specialization: Northern European panel paintings (14th–17th century)
- 12 years in conservation, 4 at GG

## Lab and tools
| Tool | Use | Notes |
|------|-----|-------|
| Bruker Tracer 5i | Handheld XRF | Primary elemental analysis tool |
| Ocean Insight FLAME | FORS spectrometer | Pigment identification |
| Nikon D850 + 105mm macro | Documentation photography | Also used for IR reflectography with filter swap |
| Zeiss Stemi 508 | Stereomicroscope | Cross-section examination |
| ImageJ (Fiji) | Spectroscopy image processing | Plugins: MineralogyToolkit |
| Python (pandas, matplotlib) | Data analysis | XRF and FORS batch processing |
| Zotero | Bibliography management | Synced to Obsidian via Citations plugin |

## Team
- **Dr. Ingrid Krause** — department head, approves all treatment proposals
- **Tobias Ackermann** — fellow conservator, shares the lab, works mostly on canvas paintings
- **Marta Delvaux** — KIK-IRPA Brussels, external collaborator on the Concordance project

## Processes
| Process | What it means |
|---------|---------------|
| Condition survey | Systematic examination of a group of paintings — visual, UV, IR, XRF |
| Treatment proposal | Written plan requiring Dr. Krause's sign-off before any intervention |
| Concordance entry | One recipe-to-spectrum mapping: source text → transcription → spectral match → confidence level |
