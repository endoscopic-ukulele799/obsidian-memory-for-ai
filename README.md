## 🔧 Troubleshooting Tips

### The App Won't Start

- Confirm you installed the latest version.
- Restart your computer and try again.
- Make sure Windows is updated.

### Cannot Connect to Vault

- Check you selected the correct Obsidian folder.
- Ensure the folder is not set to read-only.
- Close Obsidian while using the app to avoid conflicts.

### AI Does Not Remember Past Sessions

- Verify you have saved notes in your Obsidian vault.
- Allow time for the app to process your notes on the first run.
- Check your internet connection if cloud syncing is involved.
